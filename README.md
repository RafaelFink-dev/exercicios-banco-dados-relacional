# Exercícios – Banco de Dados Relacional

Este repositório contém scripts e exercícios desenvolvidos durante a disciplina **Banco de Dados Relacional** da Pós-Graduação em Desenvolvimento Full Stack.

## Objetivo
Registrar atividades práticas realizadas ao longo da disciplina, aplicando conceitos de lógica de programação e manipulação de dados em Python, com foco na estrutura e operação de banco de dados relacional.

## Conteúdos abordados
- Estruturas de controle e dados em Python
- Simulação de regras de negócio envolvendo manutenção de registros
- Scripts de manipulação e atualização de dados
- Separação de lógica e organização de código

## Tecnologias utilizadas
- Python 3

## Contexto acadêmico
Os exercícios apresentados foram desenvolvidos como parte das atividades da disciplina de Banco de Dados Relacional no contexto da Pós-Graduação em Desenvolvimento Full Stack.
